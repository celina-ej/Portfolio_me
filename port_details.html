Keyframes


Keyframes help us add animations.It gives you more control over the animation you want to perform. The animation is created by gradually changing from one style to another

Given below is the code used for animation 

@keyframes photos {
    from {
        background-color: #c8a19c;
    }
    to {
        background-color: #B2ABB3;
    }
}

Explanation


it changes the colour from #c8a19c to #B2ABB3 in a gradient manner.
When we call the keyframe name in the class we mention the duration of the animation. The sytax is as follows : 

animation: name duration timing-function delay iteration-count direction fill-mode;

The code :



            .rbox {
    position: relative;
    width: 400px;
    height: 250px;
    background-color: #B2ABB3;
    margin-left: 780px;
    bottom: 400px;
    z-index: 2;
    animation-name: photos;
    animation-duration: 4s;
    animation-iteration-count: infinite;
}
